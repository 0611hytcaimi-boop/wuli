## 使用方式

### 首页导航

打开 `index.html`，通过**左侧边栏**或**欢迎页分类卡片**选择实验，点击后跳转到对应的独立实验页面。

### 独立实验页面

每个实验有独立的 `experiments/xxx.html` 页面，可直接访问。每个页面包含：
- **顶部导航栏**：返回首页 + 其他实验快捷链接
- **Canvas 画布**：物理仿真可视化
- **参数控制面板**：滑块/按钮调节实验参数
- **实验信息面板**：实时显示物理量数据

### AI 问答

首页内置 **AI 物理助手**（由 DeepSeek API 驱动），可直接提问物理问题。API 密钥已内置，无需额外配置。

## 部署到 GitHub Pages

1. 将本项目推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Branch 选择 `master`，文件夹选择 `/ (root)`
4. 等待 1-2 分钟即可访问

## 技术栈

- 原生 JavaScript（无框架依赖）
- HTML5 Canvas 绘制
- CSS3 手绘风格 UI
- DeepSeek API（AI 问答）
